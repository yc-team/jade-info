jade-info
=========

show some info about jade


#### 工具

* 推荐：[online official](http://jade-lang.com/)
* [online html2jade](http://html2jade.aaron-powell.com/)



#### 标签

```shell
html
```

会转换为：

```shell
<html></html>
```


#### 文本

> 如何在标签里面加上文本

```shell
p 我是yc-team
```

会转换为：

```shell
<p>我是yc-team</p>
```
